# ZZM7ZIP

Plug-In Name: ZZM 7Zip
Plug-In ID: ZZM7ZIP
Description: 7-Zip is a file archiver with a high compression ratio.

Operations:

- List Archive Content
- Compress File(s)
- Extract with full path
- Extract without path
- Calculate hash
- Calculate MD5 hash
- Integrity Test
- Remove Archive
